## How to Export as PDF

1. Open `cv.html` in your browser (Chrome or Safari recommended)
2. Press `Cmd + P` to open the Print dialog
3. Set **Destination** to `Save as PDF`
4. Set **Margins** to `None` or `Minimum`
5. Enable **Background graphics** if colors appear missing
6. Click **Save**
